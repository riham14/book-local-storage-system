- This Book Application works as a database storage application for books.
- Books include a title, author, and description, along with an automatically incremented book id.
- User can add, view, search and edit books.
- The command prompt gives available choices of the system.
- To run the application:
    1- Navigate to the application folder where you're in the directory of the pom.xml file
    2- from the cmd, build the maven project by running the command 'mvn clean package'
    3- navigate to the application jar file found inside the 'target' directory,
    then run
        >> java -jar <jarName>.jar
- The user is then prompted for input.
